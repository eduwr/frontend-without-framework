## [Repositório](frontend-without-framework)

## Pré-requisitos

- [Node](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/) ou [npm](https://www.npmjs.com/get-npm)
- [Gulp](https://gulpjs.com/)

## Iniciando

Instalar as dependências com:

### `npm i`

Iniciar o projeto com o comando:

### `gulp` ou `npm start`
